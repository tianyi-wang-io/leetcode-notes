# Intuition
BFS always finds nearest path.

# Complexity
- Time complexity: $(m * n)$

- Space complexity: $(m * n)$

# Code
```python3 []
class Solution:
    def wallsAndGates(self, rooms: List[List[int]]) -> None:
        """
        Do not return anything, modify rooms in-place instead.
        """
        INF = 2147483647
        directions = [(0, 1), (1, 0), (0, -1), (-1, 0)]
        m, n = len(rooms), len(rooms[0])

        queue = deque()
        for i in range(m):
            for j in range(n):
                if rooms[i][j] == 0:
                    queue.append((0, i, j))
        
        while queue:
            distance, x, y = queue.popleft()
            
            for ix, iy in directions:
                x_next, y_next = x + ix, y + iy
                if not 0 <= x_next < m or not 0 <= y_next < n:
                    continue
                if rooms[x_next][y_next] == INF:
                    rooms[x_next][y_next] = distance + 1
                    queue.append((distance + 1, x_next, y_next))
```
